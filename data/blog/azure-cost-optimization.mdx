# Azure Cost Optimization with Budgets and Tags

Managing cloud costs is a critical part of any cloud architecture. In this project, I implemented Azure cost monitoring and optimization techniques.

## Tools Used

The following Azure services were used:

- Azure Cost Management
- Budgets and Alerts
- Resource Tagging
- Cost Analysis Dashboard

## Budget Configuration

Budgets were created to monitor monthly spending. Alerts were configured to notify when spending reached specific thresholds.

## Resource Tagging

Tags were applied to resources such as:

- Environment (Dev / Test / Production)
- Project
- Owner

This allows better tracking of costs across teams and projects.

## Cost Analysis

Azure Cost Analysis dashboards were used to monitor:

- Resource usage
- Spending trends
- Cost distribution by service

## Outcome

This implementation demonstrates how organizations can monitor and control Azure cloud spending effectively using built-in Azure cost management tools.
