# Azure Virtual Machine Infrastructure Deployment

In this project, I deployed a scalable Azure Virtual Machine infrastructure to simulate a real-world cloud environment.

## Infrastructure Components

The deployment included:

- Azure Resource Group
- Azure Virtual Network (VNet)
- Subnets
- Network Security Groups (NSG)
- Azure Virtual Machine
- Azure Monitoring

## Deployment Steps

### Resource Group
A dedicated resource group was created to manage all infrastructure resources together.

### Virtual Network
A virtual network was configured with subnets to isolate workloads and manage network traffic.

### Network Security Groups
NSG rules were implemented to allow secure inbound and outbound traffic to the virtual machine.

### Virtual Machine
A Linux-based virtual machine was deployed to host applications and simulate real cloud workloads.

### Monitoring
Azure Monitor was enabled to track performance metrics and system health.

## Outcome

This deployment demonstrates how to build secure and scalable compute infrastructure in Microsoft Azure.
